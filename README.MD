# 🤖 Job Role Predictor

A Machine Learning and NLP based web application that predicts the most suitable job role from the text of a resume.

## 📌 About the Project

This project was built as part of my Machine Learning and Natural Language Processing (NLP) learning journey.

The application allows users to paste resume content into a web page and predicts the most relevant job role based on the skills and information present in the resume.

## ▶️ Run Locally

Clone the repository

```bash
git clone https://github.com/navyakhanna2020/Job-Role-Predictor.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
